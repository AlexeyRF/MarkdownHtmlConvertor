```bash
pip install markdown

```
##  Использование

Скрипт предоставляет две функции: `mdcode` для строк и `mdfile` для файлов.

### 1. Конвертация текста

Используйте `mdcode`, для создания из markdown кусочка html. На вход принимает строку.

### 2. Конвертация файла

Используйте `mdfile` для создания HTML-файла. <br>Параметры: md_path (путь к исходному md файлу) out_path (путь сохранения) use_css (Использовать ли CSS) 
css_path (если прошлый параметр True, то просит указать путь к css, если None, то используется предустановленный стиль) inline_css (CSS код встраивается внутрь HTML через <style>?)
